# Workout API

## 📌 Project Description
A Flask REST API for tracking workouts and exercises. Each workout can contain multiple exercises with sets, reps, and duration using a join table.

---

## ⚙️ Technologies Used
- Python
- Flask
- Flask SQLAlchemy
- Flask Migrate
- Marshmallow
- SQLite

---

## 📁 Project Structure
